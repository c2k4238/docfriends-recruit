# 홈페이지 과제 경로(Vue.js)
Front > Project > project_doc

webpack-cli을 이용하여 vue.js 프로젝트를 구성하였습니다.
UI는 Vuetify 홈페이지를 참고하여 구성하였습니다.

사용된 라이브러리는 다음과 같습니다.
  "axios"
  "es6-promise"
  "vue"
  "vue-lodash"
  "vue-router"
  "vue2-google-maps"
  "vuetify"
  "vuex"

# localhost:8080 or ip:8080 웹 서비스 실행
 -- npm run dev
